# Comparaison d'Architectures CNN et Transfer Learning

*Vision artificielle et traitement des images - UQAC Hiver 2026*

## 🚀 Description du projet

L'objectif principal est de concevoir et de comparer deux approches de classification d'images:
- Transfer Learning : Adaptation d'une architecture pré-entraînée connue 
- CNN Custom : Création et entraînement d'une architecture de réseau de neurones convolutifs propre

## 📂 Lancement du projet

### Prérequis

- Assurez-vous d'avoir Python installé
- Installez les dépendances requises :  
    ```bash
    pip install -r requirements.txt
    ```

### Préparation des données

Nous avons utilisé le dataset Butterfly Image Classification (https://www.kaggle.com/datasets/phucthaiv02/butterfly-image-classification) pour l'entraînement et l'évaluation des modèles. Le script `data_preparation.py` s'occupe de télécharger et de prétraiter les données.

### Exécution du projet

Pour lancer l'entraînement et l'évaluation des modèles, exécutez le script principal :
```bash
python TP3.ipynb
```
